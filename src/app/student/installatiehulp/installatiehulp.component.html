Installatiehulp

De officiële installatie instructies staan op deze webpagina.


JDK downloaden

JDK installeren:
- Download JDK
- Voer het installatie bestand uit en volg de instructies

Aanpassen van het PATH (of Path) bij SystemProperties/Systeemeigenschappen:
- Druk op Windows toets + R
- In de tekst box die nu verschijnt type je in: SystemPropertiesAdvanced
- Een nieuw window opent in de tab: Advanced / Geavanceerd
- Selecteer: Environment Variable / Omgevingsvariabelen
- In System Variables / Systeemvariabelen: selecteer de PATH variabele om te bewerken
- Voeg toe aan einde van de PATH variabele de locatie van de bin directory van de JDK, bv:   ;C:\Program Files\Java\jdk1.8.0_66\bin
  Let op: de ; dient als scheidingsteken met het bestaande PATH

Installatie testen
- Druk op Windows toets + R
- In de tekst box die nu verschijnt type je: cmd
- Een nieuw command window opent, type hierin: java -version

De output die je nu krijgt zal er als volgt uitzien (nummers verschillen afhankelijk van de geïnstalleerde versie):

Java(TM) SE Runtime Environment (build 1.8.0_91-b14)
Java HotSpot(TM) 64-Bit Server VM (build 25.91-b14, mixed mode)
Indien de installatie niet correct is zie je als output:
java wordt niet herkend als een interne of externe opdracht, programma of batchbestand.
