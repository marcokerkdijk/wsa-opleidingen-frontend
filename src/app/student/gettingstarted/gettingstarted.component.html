Getting Started
Als je dit leest ben je voornemens om programmeur te worden. Onze tip is om meteen te starten om je te bekwamen in programmeren. Working Spirit heeft een selectie gemaakt uit het brede aan bod van online cursussen en tutorials. Je zult merken dat wanneer je deze tutorials gedaan hebt, je de concepten die je in het Working Spirit traject worden uitgelegd, makkelijker kunt volgen.
Let op! Volg de cursussen over Java, niet JavaScript. Dat is namelijk een significant andere taal.
Codecademy
De website van Codecademy is ideaal om mee te beginnen: www.codecademy.com Het neemt je met behulp van een intelligente website mee in de beginselen van Java. Je leert wat een boolean, een variabele en een functie is. Heel basaal, maar zeker leuk. Doe deze als eerste.
The New Boston
Hier heb je de cursus Java - beginners - The New Boston beginners Tijdens deze met name video tutorial krijg je een goed overzicht in de beginselen van Java en programmeren. Ook leren met name eerste video's je, hoe je jouw computer geschikt moet maken voor het programmeren met Java.
Moocfi
Iets meer tekstueel en met goede codevoorbeelden is: moocfi
Tutorialspoint
Ook tutorialspoint start met een hele mooie introductie op de taal Java: tutorialspoint
Tot slot
Tot slot zou het handig zijn als je Java alvast installeerd op je computer. Volg deze stappen om je computer goed te prepareren voor de Java-cursus:
- Download en installeer JDK
- Download en installeer Eclipse
- Maak je eerste programma: Hello World.

Succes en veel plezier toegewenst!
